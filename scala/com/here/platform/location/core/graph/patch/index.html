  2.  trait VirtualIndexCodec[V, E] extends VirtualVertexCodec[V]

    A codec that encodes and decodes virtual vertices and edges within the concrete vertex/edge types V and E of a patched graph.

    A codec that encodes and decodes virtual vertices and edges within the concrete vertex/edge types V and E of a patched graph.

    Virtual vertices and edges are synthetic graph elements introduced by a patch — they exist in the patched graph but not in the original. Each virtual vertex is identified by a zero-based virtualVertexIndex and each virtual edge by a zero-based virtualEdgeIndex.

    Implementations pack these indices inside the existing types V and E (for example by using a reserved tile id), avoiding new type allocations while still allowing virtual elements to be distinguished from original ones at runtime.

    V

    The vertex type of the graph.

    E

    The edge type of the graph.

    Annotations
    @Experimental()
    See also

    VirtualVertexCodec for the vertex-only subset of this interface.
