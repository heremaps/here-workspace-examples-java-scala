    Custom decoder for forbiddenHazardousGoods field in VehicleRestriction.

    Custom decoder for forbiddenHazardousGoods field in VehicleRestriction. The Router API v8 can return this field in two formats: 1. As a string: "any" 2. As an array: ["gas", "explosive", "flammable"]

    This decoder handles both by converting string to single-element Seq.
