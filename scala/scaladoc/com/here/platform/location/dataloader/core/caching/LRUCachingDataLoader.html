com.here.platform.location.dataloader.core.caching

LRUCachingDataLoader 

class LRUCachingDataLoader[Id, T] extends AbstractDataLoaderDecorator[Id, T] with LRUCachingDataLoaderImpl[Id, T]

A common implementation for different data loaders that can be wrapped in an LRU cache.

The cache size is limited to the given maximum size (in number of elements).

Id

The id type used to identify the partition, i.e. tile ids or generic ids.

T

The type of data loaded.

Annotations

Deprecated

(Since version SDK 2.54) Use methods based on OptimizedMapLayers instead.
