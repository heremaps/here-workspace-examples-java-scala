    Deletes one or more changesets where the version number can be specified by the query parameter 'version', where the value must be an integer, bigger than 0 and the comparator should be one of the below list: - < - =lt= Example: To remove changesets older than version number 10, a DELETE /layers/{layerId}/changesets?version=lt=10 could be executed.   10.  def getChangeset(hrn: String, layerId: String, version: String, pageToken: Option[String] = None, limit: Option[Int] = None): ApiRequest[Changeset]

    Retrieves one Changeset from the layer by version.

    Retrieves one Changeset from the layer by version. Each successful write transaction to the layer is stored as one single Changeset, which can contain modifications applied to one or more features. The response payload may be split in multiple pages, next page is written in the property 'nextPageToken', which then, can be retrieved by using the 'pageToken' parameter.

    Path: /layers/{layerId}/changesets/{version}

    Method: GET

    Expected answers:

    code 200 : Changeset (One Changeset which contains FeatureCollections grouped by operation.) Headers : X-Correlation-ID - X-Request-ID -

    code 401 : Error (Unauthorized to perform the request.)

    code 513 : Error (Response payload too large.) Headers : X-Correlation-ID - X-Request-ID -

    hrn

    The HERE Resource Name (HRN) of the catalog.

    layerId

    The unique identifier of the layer.

    version

    The version of a Changeset.

    pageToken

    The page token where the iteration will continue.

    limit

    The maximum number of features in the response. Default is _30000_. Hard limit is _100000_.
