  6.  def deleteArtifact(hrn: String, artifactHrn: String, force: Option[Boolean] = None): ApiRequest[DeleteArtifactResponse]

    Delete the artifact and related files by given HRN.

    Delete the artifact and related files by given HRN. If scoped token is used, then the artifact is deleted within the project the token belongs to. Partial HRN without version could be used. If more than one version of artifact exists and partial HRN is passed then force flag must be set to true. By default force flag is set to false.

    Path: /artifact/{artifactHrn}

    Method: DELETE

    Expected answers:

    code 200 : DeleteArtifactResponse (Successfully deleted existing artifact)

    code 400 : (Invalid artifact details were provided)

    code 401 : (You are not authorized to view the artifact or have wrong / invalid credentials)

    code 403 : (Accessing the artifact is forbidden)

    code 404 : (Specified artifact not found)

    code 500 : ErrorResponse (Internal server error. Support ticket should be created)

    hrn

    The HERE Resource Name (HRN) of the catalog.

    artifactHrn

    The HRN of the artifact

    force

    The flag to force the deletion even if the artifact is linked to a project

  18.  def listArtifacts(access: Option[String] = None, sort: Option[String] = None, order: Option[String] = None, from: Option[String] = None, groupId: Option[String] = None, artifactId: Option[String] = None, offset: Option[Long] = None, limit: Option[Int] = None): ApiRequest[ListArtifactsResponse]

    Return the list of available artifacts.

    Return the list of available artifacts. Requires permission to 'apiAccess'.If scoped token is used, then the list of artifacts is returned within the project the token belongs to.If unscoped token is used, then the list of artifacts is returned including the artifacts belonging to the projects.

    Path: /artifact

    Method: GET

    Expected answers:

    code 200 : ListArtifactsResponse (OK)

    code 401 : (You are not authorized to view the artifact or have wrong / invalid credentials)

    code 403 : (Accessing the artifact is forbidden)

    code 400 : (Invalid artifact list parameters were provided)

    code 500 : ErrorResponse (Internal server error. Support ticket should be created)

    access

    Comma separated list of access types (with OR semantic) to filter result.If 'default', returns artifacts user has read access to. If 'orgAdmin', returns artifacts OrgAdmin user has access to.

    sort

    Applies sorting based on specified fields. Allowed values are group, name, version. Multiple fields must be separated with ':'

    order

    Applies ordering. Defaults is ASC

    from

    Applies 'greater than' filtering to fields specified in 'sort' property. For example when 'sort' property is 'name:version' and 'from' is 'b:1.0.0' then the response will contain records with a name starting from 'b' till 'z' and version greater than 1.0.0

    groupId

    Applies filtering based on groupId

    artifactId

    Applies filtering based on artifactId

    offset

    Specifies how many records will be skipped at the beginning. Default is 0

    limit

    Specifies the maximum amount of records that could be returned. Default is 20
